# todoMVC-tag
An implementation of an entire todoMVC application in a single tag.

##Installation
Just include the script in your project.
```
<script src="todo-mvc.js"></script>
```

##Usage
```
<todo-mvc /> //That's it!
```

##Running example

```
$ cd path/to/repo
$ python -m SimpleHTTPServer 8080
$ open http://localhost:8080/example
```
